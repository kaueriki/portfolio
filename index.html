A solução será composta pelos seguintes elementos:​ Desenvolvimento das Estações Meteorológicas, transmissão de Dados, pprocessamento e armazenamento em Servidor

Semestre - 2025-1

Empresa parceira: Pro4Tech

Problema: Este projeto tem como objetivo o desenvolvimento de uma plataforma de criação e gerenciamento de agentes de Inteligência Artificial. A ferramenta permitirá que administradores criem múltiplos agentes virtuais personalizados, treinados para responder dúvidas específicas sobre diferentes sistemas e processos internos da empresa. Além disso, a solução deve contar com um sistema de permissões, onde cada usuário terá acesso apenas aos agentes aos quais foi autorizado.

Solução: Foi desenvolvido um aplicativo móvel que possibilita a interação com esses agentes por meio de um chat inteligente, garantindo que cada usuário visualize apenas as conversas e respostas dos agentes aos quais possui permissão de acesso. A aplicação é integrada a um banco de dados na nuvem para armazenamento das interações e permitir o gerenciamento eficiente dos agentes e permissões.

Contribuições Pessoais:

Neste projeto, atuei como Desenvolvedor Frontend, realizei o desenvolvimento das telas de Login, Perfil, Curador e Administrador. Implementei as funções do Backend no Frontend de CRUD dos Agentes e Usuários, e também as permissões de quais usuários pode conversar com os agentes específicos.

Hard Skils:

Desenvolvimento Frontend em React-native e integrar o Backend com o Frontend em um Emulador Android pelo Android Studio.

Soft Skils:

- Comunicação

- Trabalho em equipe

- Entrega eficiente

- Gestão de Tempo e Prioridades

Tecnologias utilizadas: React-native, HTML, CSS, TypeScript, C#, DotNet, Gemini Redis e MySQL.
